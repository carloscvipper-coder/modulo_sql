## EMT

EMT Madrid necesita una base de datos para registrar y gestionar la información de las **líneas de transporte de Madrid**.

Es necesario registrar las **líneas** de transporte que se realizan en Madrid, los **conductores** que las realizan y el tiempo gastado para realizar cada línea.

1. Crea la base de datos **emt** en MySQL.
2. Crea las tablas **lineas** y **conductores**. Usa los atributos, tipos de datos y restricciones que consideres necesarios para cada table. **No se usarán auto incrementales**.
3. Inserta un mínimo de 10 registros en cada tabla.
4. Plantea como mínimo 5 consultas de información con `SELECT`, filtrando información y haciendo uso de las palabras clave `WHERE`, `ORDER BY`, `JOIN`, `UPDATE`.
